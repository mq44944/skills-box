## 1. shell编程之变量
1. BASH中变量类型默认为字符串型  
2. 变量分类：用户自定义变量、环境变量、位置参数变量、预定义变量  

export 可以把自定义变量声明成为环境变量  
set 查看所有变量   
env 查看环境变量
